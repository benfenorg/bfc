# Getting Started

### Installation

To install the
[React Sui Wallet Kit library from npm](https://www.npmjs.com/package/@benfen/wallet-kit), along
with the Sui TypeScript SDK:

```bash
npm install @benfen/wallet-kit @benfen/bfc.js
```

### Configuring the Sui Wallet Kit Provider

After you install the `@benfen/wallet-kit` package, you need to wrap your application with the
`WalletKitProvider`.

```tsx
import { WalletKitProvider } from '@benfen/wallet-kit';

function App() {
  return (
    <WalletKitProvider>
      <YourApp />
    <WalletKitProvider />
  );
};
```

### Rendering a Connect Button

After you wrap your app with the `WalletKitProvider`, add a `ConnectButton` component that users can
click to connect their wallet to your dApp.

```tsx
import { ConnectButton } from '@benfen/wallet-kit';

export function YourApp() {
	return <ConnectButton />;
}
```
